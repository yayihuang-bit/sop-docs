# 競品活動分析系統 — 使用指南

## 我想做什麼？

| 目的 | 做法 |
|------|------|
| 📖 閱讀使用指南 | 打開 `使用指南.html` |
| ✏️ 修改文字內容 | 修改 `使用指南.md`，再找 Claude 同步 |
| 🖼️ 更新圖片或影片 | 找 Claude 協助 |

---

## 資料夾說明

| 檔案／資料夾 | 說明 |
|------------|------|
| `使用指南.html` | 分享用，圖片影片全部嵌入，可單獨傳給別人 |
| `使用指南.md` | 文字內容主檔，維護時改這個 |
| `示意圖/` | 使用指南的說明圖片 |
| `參考影片/` | 使用指南的教學影片 |
| `_維護/` | 維護用腳本與原始檔（一般不需要進去） |
| `系統程式碼/` | 競品分析系統的 GAS 程式碼 |
| `參考文件/` | 架構書、優化建議等參考資料 |
