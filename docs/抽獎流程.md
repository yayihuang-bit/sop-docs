## 執行流程

1. 營運窗口確認 [活動時程](https://docs.google.com/spreadsheets/d/1B9F0DOBFTuBSmCbz3zaJ5ksPCbh6PJNAPoh-jsonVyQ/edit?gid=1985174151#gid=1985174151)，將[抽獎](https://docs.google.com/spreadsheets/d/1QLXHs-ObTqAVVp1uwm841Lcsq3lhQAXJJslqW5wVq84/edit?gid=1414618376#gid=1414618376)與[換皮](https://yayihuang-bit.github.io/sop-docs/%E6%8F%9B%E7%9A%AE%E8%B3%87%E8%A8%8A_%E7%B4%A2%E5%BC%95/)項目排入時程

2. 營運企劃開始安排[獎項內容](https://docs.google.com/spreadsheets/d/1QLXHs-ObTqAVVp1uwm841Lcsq3lhQAXJJslqW5wVq84/edit?gid=1414618376#gid=1414618376)、[換皮](https://yayihuang-bit.github.io/sop-docs/%E6%8F%9B%E7%9A%AE%E8%B3%87%E8%A8%8A_%E7%B4%A2%E5%BC%95/)內容初版

3. 與[行銷窗口](https://docs.google.com/presentation/d/1aGmy3_ZOKVrzxiTna8NUukvYr2gb10t_BQwuaEcMd2U/edit?slide=id.g3b3122b0bb8_0_51#slide=id.g3b3122b0bb8_0_51)討論換皮與獎項規劃

4. 與營運、數學窗口討論做初版定案討論

5. 將獎項總預算、規模、各項目提供給米乓審查

6. 同步行銷結論，提供活動主副標做最終確認

7. 製作活動文件

8. 釋出至 Slack [包你發營運組](https://yiletechnology.slack.com/archives/C0AH47UFFJQ)
